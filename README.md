# Tredence-Assignment-HR-Workflow-Designer-Module# HR Workflow Designer - Prototype

## Run
1. npm install
2. npm run dev
3. Open http://localhost:5173

## What is included
- Vite + React + TypeScript
- React Flow canvas with 5 node types
- Sidebar drag & drop
- Node configuration forms
- MSW mock API (GET /automations, POST /simulate)
- Sandbox panel to run simulation + validation
